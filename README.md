# DZ_8_Docker

Домашняя работа № 1.

1. Описать Dockerfile для запуска контейнера с проектом.
2. Обернуть в Docker Compose Django-проект с БД PostgreSQL.
3. Дописать в docker-compose.yaml работу с Redis.
4. Дописать в docker-compose.yaml работу с Celery.

Запуск проекта:

1. На рабочей станции установлен Docker Desktop
2. Клонировать репозиторий на свою рабочую станцию
3. Создать файл .env в котором указать данные для переменных окружения(шаблон файла .env.sample)
4. Комнада для сборки образа: docker-compose build
5. Команда для запуска проекта: docker-compose up
